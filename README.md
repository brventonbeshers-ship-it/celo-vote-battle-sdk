# celo-vote-battle-sdk

TypeScript SDK for interacting with the CeloVoteBattle contracts on Celo.

## Installation

```bash
npm install celo-vote-battle-sdk
```

## Usage

```ts
import { VoteBattleClient } from "celo-vote-battle-sdk";

const client = new VoteBattleClient({
  contractAddress: "0x..."
});
```

## Development

```bash
npm install
npm run build
```
